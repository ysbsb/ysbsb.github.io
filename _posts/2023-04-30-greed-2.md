# 1. Greedy algorithm

## ë°±ì¤€ 1931ë²ˆ



https://www.acmicpc.net/problem/1931



```python
N = int(input())
A = [[0]*2 for _ in range(N)]

for i in range(N):
    S, E = map(int, input().split())
    A[i][0] = E
    A[i][1] = S
    
A.sort()
count = 0
end = -1

for i in range(N):
    if A[i][1] >= end:
        end = A[i][0]
        count += 1
        
print(count)
```

## ë°±ì¤€ 1541ë²ˆ



https://www.acmicpc.net/problem/1541



```python
answer = 0
A = list(map(str, input().split("-")))

def mySum(i):
    sum = 0
    temp = str(i).split("+")
    for i in temp:
        sum += int(i)
    return sum

for i in range(len(A)):
    temp = mySum(A[i])
    if i==0:
        answer += temp
    else:
        answer -= temp
        
print(answer)
```
